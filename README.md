## Installation

Install packages with [`npm`](https://www.npmjs.com/) or [`yarn`](https://yarnpkg.com/en/):


  ```bash
$ yarn build
$ yarn start
  ```
  
or

```bash
$ npm install
```

Then, start the project!
  
  ```bash
$ yarn build
$ yarn start
  ```
  
or

  ```bash
$ npm build
$ npm start
  ```
